📱 تنظیمات گوشی؛ هر چیزی که باید بدانید

گوشی‌های هوشمند امروزی پر از تنظیمات مختلف هستن. از وای‌فای و اینترنت گرفته تا امنیت و حریم خصوصی. تو این راهنما، قدم به قدم تمام تنظیمات مهم رو بهتون آموزش می‌دیم.

نمایی از منوی تنظیمات گوشی‌های اندروید

فرقی نمی‌کنه گوشیتون سامسونگ باشه یا شیائومی، آیفون یا هر برند دیگه‌ای. اصول کلی تنظیمات یکسانه و ما اینجا همه رو پوشش می‌دیم.

📶 ۱. تنظیمات اینترنت و وای‌فای

📡 اتصال به وای‌فای

  1. وارد Settings (تنظیمات) بشید
  2. روی Wi-Fi یا Network & Internet کلیک کنید
  3. وای‌فای رو روشن کنید
  4. شبکه مورد نظر رو انتخاب کنید
  5. رمز عبور رو وارد کنید
💡 نکته: اگه وای‌فای وصل نمیشه، مودم رو یه بار خاموش و روشن کنید. معمولاً مشکل حل میشه!

📱 اینترنت سیم‌کارت (دیتا)

برندمسیر تنظیمات
سامسونگSettings → Connections → Data Usage → Mobile Data
شیائومیSettings → SIM cards & mobile networks → Data
آیفونSettings → Cellular → Cellular Data
اندروید خامSettings → Network & Internet → Mobile Network

🔋 ۲. تنظیمات باتری و صرفه‌جویی

⚡ حالت ذخیره باتری (Power Saving)

همه گوشی‌ها یه حالت صرفه‌جویی در مصرف باتری دارن که وقتی باتری کمه، به‌طور خودکار برنامه‌های پس‌زمینه رو محدود می‌کنه.

برندمسیر فعال‌سازی
سامسونگSettings → Battery → Power Saving
شیائومیSettings → Battery → Battery Saver
آیفونSettings → Battery → Low Power Mode
⚠️ هشدار: استفاده طولانی از حالت ذخیره باتری ممکنه باعث بشه بعضی اعلان‌ها (Notifications) رو دریافت نکنید.

🎥 ویدیوی آموزشی

🔒 ۳. تنظیمات امنیتی

🔑 قفل صفحه (Screen Lock)

انواع قفل صفحه که می‌تونید استفاده کنید:

  • Pattern (الگو) — کشیدن شکل روی ۹ نقطه
  • PIN — کد ۴ تا ۶ رقمی
  • Password — رمز ترکیبی از حروف و اعداد
  • Fingerprint (اثر انگشت) — سریع‌ترین روش
  • Face Unlock (تشخیص چهره) — مخصوص گوشی‌های جدید
💡 توصیه: بهترین کار استفاده از اثر انگشت + PIN هست. هم سریع باز میشه، هم امنیت بالایی داره.

🛡️ گوگل پلی پروتکت

Google Play Protect یه سرویس امنیتی رایگان از گوگله که برنامه‌های نصب‌شده رو اسکن می‌کنه و بدافزارها رو شناسایی می‌کنه.

مسیر: Google Play Store → Menu → Play Protect

📡 ۴. تنظیمات بلوتوث

🔗 اتصال به دستگاه بلوتوثی

  1. بلوتوث دستگاه مورد نظر (مثلاً هندزفری) رو روشن کنید
  2. توی گوشی: Settings → Bluetooth
  3. بلوتوث رو روشن کنید
  4. از لیست، دستگاه رو پیدا و انتخاب کنید
  5. روی Pair کلیک کنید
💡 نکته: اگه دستگاه پیدا نشد، یه بار بلوتوث رو خاموش و روشن کنید. مطمئن بشید دستگاه در حالت Pairing قرار داره.

🖥️ ۵. تنظیمات نمایشگر

💡 روشنایی (Brightness)

کشیدن نوار اعلان از بالای صفحه به پایین، سریع‌ترین راه تنظیم روشناییه. همچنین می‌تونید از مسیر Settings → Display → Brightness اقدام کنید.

🌙 حالت تاریک (Dark Mode)

برندمسیر
سامسونگSettings → Display → Dark Mode
شیائومیSettings → Display → Dark Mode
آیفونSettings → Display & Brightness → Dark
💡 فایده Dark Mode: هم باتری کمتر مصرف میشه (مخصوص نمایشگرهای OLED)، هم چشم کمتر خسته میشه.

حالت تاریک در گوشی‌های مختلف

📦 ۶. مدیریت برنامه‌ها

🗑 حذف برنامه

  1. Settings → Apps
  2. برنامه مورد نظر رو پیدا کنید
  3. روش کلیک کنید
  4. Uninstall رو بزنید

🧹 پاک کردن کش (Cache)

کش برنامه‌ها فایل‌های موقتی هستن که به مرور زمان فضای زیادی اشغال می‌کنن. پاک کردنشون آسیبی به اطلاعات برنامه نمی‌زنه.

مسیر: Settings → Apps → [اسم برنامه] → Storage → Clear Cache

⚠️ دقت کنید: گزینه Clear Data اطلاعات برنامه رو کامل پاک می‌کنه. فقط زمانی ازش استفاده کنید که برنامه مشکل جدی داره.

👤 ۷. حساب‌های کاربری

➕ اضافه کردن حساب گوگل

  1. Settings → Accounts (یا Users & Accounts)
  2. Add Account
  3. Google رو انتخاب کنید
  4. ایمیل و رمز عبور رو وارد کنید

با اضافه کردن حساب گوگل، می‌تونید از Gmail، Google Drive، Google Photos و کلی سرویس دیگه استفاده کنید.

🔄 ۸. بازنشانی (Reset)

🔁 Factory Reset (بازگشت به تنظیمات کارخانه)

⚠️ بسیار مهم: این کار تمام اطلاعات گوشی رو پاک می‌کنه! قبلش حتماً از عکس‌ها، مخاطبین و فایل‌هاتون پشتیبان بگیرید.
برندمسیر
سامسونگSettings → General Management → Reset → Factory Data Reset
شیائومیSettings → About Phone → Factory Reset
آیفونSettings → General → Transfer or Reset iPhone → Erase All Content

بعد از بازنشانی، گوشی مثل روز اول میشه و باید دوباره تنظیمش کنید.
