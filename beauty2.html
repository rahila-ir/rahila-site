🔙 زیبایی ۱

۲۱. 💇 کراتینه مو

📋 توضیح

صاف و ابریشمی کردن موها. ماندگاری ۳-۶ ماه. موها رو نرم و براق می‌کنه.

⭐ نکات

  • تا ۳ روز نشور
  • شامپو بدون سولفات
  • ورزش سنگین نکن

۲۲. 👁️ میکروبلیدینگ ابرو

📋 توضیح

تاتوی نیمه‌دائم ابرو. ماندگاری ۱-۲ سال. ظاهر طبیعی و پرپشت.

۲۳. 💄 کانتور صورت

📋 روش

تیره کردن قسمت‌های فرو رفته (زیر استخوان گونه، کنار بینی). روشن کردن قسمت‌های برجسته (بالای گونه، وسط پیشانی).

۲۴. 🧴 روتین مراقبت از مو

📋 مراحل

۱. شامپو ۲. نرم‌کننده ۳. ماسک مو (هفته‌ای ۱-۲ بار) ۴. روغن مو ۵. محافظ حرارت قبل سشوار.

۲۵. 💅 ژلیش ناخن

📋 توضیح

لاک ژل که با UV خشک میشه. ماندگاری ۲-۳ هفته بدون خش. براق و مقاوم.

۲۶. 💆 فیشیال صورت

📋 مراحل

پاکسازی → بخور → لایه‌برداری → ماسک → مرطوب‌کننده. ماهی ۱ بار توصیه میشه.

۲۷. 👁️ خط چشم گربه‌ای

📋 روش

از گوشه داخلی چشم نازک شروع کن، به سمت خارج ضخیم‌تر کن، دمش رو به سمت بالا بکش.

۲۸. 💇 بابلیس و حالت مو

📋 روش

موها رو به چند قسمت تقسیم کن. هر دسته رو دور بابلیس بپیچ. ۱۰-۱۵ ثانیه نگه دار. با اسپری فیکس کن.

۲۹. 🧴 ضد لک

📋 مواد مؤثر

ویتامین C، نیاسینامید، آربوتین، رتینول، اسید کوجیک. ضدآفتاب الزامیست.

۳۰. 👄 رژ لب مات

📋 روش

لب رو لایه‌برداری کن. مرطوب‌کننده بزن. خط لب بکش. رژ لب مات بزن. با دستمال اضافه رو بگیر.

۳۱. 💆 مزوتراپی مو

📋 توضیح

تزریق ویتامین و مواد مغذی به پوست سر. جلوگیری از ریزش و تقویت رشد مو. ۶-۸ جلسه نیاز داره.

۳۲. 🧴 آبرسان پوست

📋 انواع

ژل (چرب)، کرم (معمولی)، لوسیون (خشک). صبح و شب استفاده کن.

۳۳. 💅 ناخن مصنوعی

📋 انواع

کاشتنی، قالبی، نواری. سریع و آسون. برای موقعیت‌های خاص.

۳۴. 👁️ مژه مصنوعی

📋 انواع

نواری، دسته‌ای، مگنتی. برای استفاده روزانه یا مهمانی.

۳۵. 💇 رنگ موی گیاهی

📋 انواع

حنا (قرمز-نارنجی)، وسمه (مشکی-قهوه‌ای)، بابونه (طلایی)، قهوه (قهوه‌ای).

۳۶. 🧴 ضد چروک

📋 مواد مؤثر

رتینول، هیالورونیک اسید، پپتیدها، ویتامین E، کلاژن.

۳۷. 💄 هایلایتر

📋 روش

بالای استخوان گونه، زیر ابرو، نوک بینی، گوشه داخلی چشم. به صورت درخشندگی میده.

۳۸. 👁️ سایه چشم

📋 انواع

مات، براق، شیمر، کرمی، پودری. رنگ‌های گرم (نارنجی، قهوه‌ای) برای چشم قهوه‌ای.

۳۹. 💇 پروتئین تراپی مو

📋 توضیح

ترمیم موهای آسیب‌دیده. پروتئین هیدرولیز شده جذب مو میشه. موها رو قوی و براق می‌کنه.

۴۰. 🧴 اسکراب بدن

📋 مواد خانگی

شکر + روغن نارگیل. قهوه + عسل. نمک دریا + روغن زیتون. هفته‌ای ۱-۲ بار.
